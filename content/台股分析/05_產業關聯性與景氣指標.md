# 📈 產業關聯性與景氣指標

> [!important] 本頁的核心價值
> 知道每家公司在供應鏈的位置，就能預測：
> - **誰先反映景氣**（先行指標）
> - **誰最後才動**（落後指標）
> - **何時切換買哪個層次**（選股切入框架）

---

## ⏱️ 景氣傳導完整時間線

### 傳統科技景氣循環（手機 / PC / 消費性電子）

```
景氣底部（庫存最高、股價最低）
    │
    ▼ 0個月
【上游訊號出現】
  • 矽晶圓廠接到新長約詢問
  • 特殊氣體/化學品訂單小增
  • 環球晶報價止跌（原本一直降價）
    │
    ▼ +1～2季
【中游訊號】
  • 台積電法說：稼動率回升至 90%+
  • IC設計大廠（聯發科/聯詠）：庫存天數 < 70天
  • 晶圓廠產能開始供不應求，需要等產能
    │
    ▼ +1～2季
【下游訊號】
  • 日月光封測廠稼動率 > 85%
  • 手機/PC品牌商出貨量年增率轉正
  • 零售通路庫存降至正常水位
    │
    ▼ 景氣高峰（股價通常已提前半年反映）
```

> [!tip] 關鍵原則
> **股價領先基本面 6-9 個月**。
> 等數據確認後才買，通常只能買到後半段漲幅。

---

### AI 科技景氣循環（伺服器 / 雲端）

```
AI 需求催化劑（ChatGPT 爆發 / 新模型發布）
    │
    ▼ 立即
【雲端廠 CapEx 上修】
  Meta / Microsoft / Google / Amazon 財報
  宣布 AI 投資擴大
    │
    ▼ +1季
【供應鏈搶單】
  廣達、緯穎接到 AI 伺服器訂單
  南電、欣興 ABF 載板交期拉長（供不應求）
  奇鋐液冷模組被指定配套
    │
    ▼ +1～2季
【台積電 CoWoS 擴產】
  台積電宣布 CoWoS 產能計畫
  上游設備廠、廠務廠商接到台積電訂單
    │
    ▼ 持續成長（非週期性，需求不會突然消失）
```

> [!tip] AI vs 傳統科技的核心差異
> | 比較項目 | 傳統科技（消費電子）| AI 科技（雲端/伺服器）|
> |---------|---------------|-------------------|
> | 景氣週期 | 2-4年一個完整循環 | 目前仍是長期成長趨勢 |
> | 庫存週期 | 明顯（囤積→去化→補庫）| 較不明顯（需求持續）|
> | 需求確定性 | 低（消費者需求難測）| 高（雲端廠多年計畫）|
> | 股價波動 | 大（景氣循環）| 大（但因預期差）|

## 📊 庫存循環圖解

### 半導體庫存週期的四個階段

```
        庫存天數（週）
        ↑
  高    │    ┌──────┐
        │   │ 去化期 │ ← 最痛苦，股價最低
        │   │（賣不動）│
        │   └──┬───┘
        │      │
        │  ┌──▼────────┐
  中    │  │  正常水位    │
        │  │（50-70天）  │
        │  └──┬──────┬─┘
        │     │      │
        │  ┌──▼─┐ ┌──▼────────┐
  低    │  │補庫  │ │ 供不應求   │
        │  │存期  │ │（缺貨漲價）│ ← 最甜蜜，股價最高
        │  └────┘ └──────────┘
        └────────────────────────→ 時間
```

### 各產業庫存正常水位

| 產業 | 正常庫存天數 | 過高警戒 | 過低缺貨 |
|------|-----------|---------|---------|
| IC 設計 | 60-80 天 | > 100 天 | < 40 天 |
| 晶圓代工 | 稼動率指標 | < 75% | > 95% |
| 封測 | 稼動率指標 | < 70% | > 90% |
| 系統廠 | 30-50 天 | > 70 天 | < 20 天 |

## 🎯 選股切入時機框架

### 框架概念

```
景氣復甦 → 景氣確認 → 景氣高峰
    ↓           ↓          ↓
  買上游      買中游     買下游
（先行指標）（確認訊號）（落後指標）
```

### 各階段選股策略

#### 📍 景氣底部期（最佳買點，最難操作）

> [!tip] 適合標的：上游材料 + 週期性IC設計

**訊號：**
- 台積電稼動率跌破 80%
- IC設計業庫存天數超過 100 天
- 矽晶圓現貨價跌至長約價 80% 以下
- 費城半導體指數年跌幅超過 30%

**建議布局：**
- 環球晶 (6488)：矽晶圓跌至成本附近
- 南亞科 (2408)：DRAM 現貨價觸底
- 聯詠 (3034)：面板驅動 IC 庫存接近出清

---

#### 📍 景氣復甦期（成功機率最高）

> [!tip] 適合標的：中游晶圓代工 + 主流 IC 設計

**訊號：**
- 台積電法說下修後首次維持展望
- IC設計廠庫存天數降回 70-80 天
- 手機出貨量年增率轉正（YoY > 0%）
- 台積電稼動率回升至 85%+

**建議布局：**
- 台積電 (2330)：確認先進製程需求回升
- 聯發科 (2454)：中國手機庫存去化完成
- 瑞鼎 (3592)：AMOLED 手機訂單回升
- 日月光 (3711)：封測稼動率回升

---

#### 📍 景氣旺季（順勢追漲，注意風險）

> [!warning] 此時買入需設好停損

**訊號：**
- 台積電連續兩季上修展望
- 缺貨漲價消息頻傳
- 科技類股本益比普遍拉高

**可留倉標的（已在手的繼續持有）：**
- 廣達 (2382)：AI 伺服器出貨高峰期
- 緯穎 (6669)：Hyperscaler CapEx 仍在加速
- 川湖 (2059)：AI 機架滑軌需求持續

### 每季追蹤（法說/財報）

| 公司/指標 | 關鍵觀察點 |
|---------|----------|
| 台積電法說 | CoWoS 稼動率、全年展望、CapEx 指引 |
| 聯發科財報 | 庫存天數、手機市場能見度 |
| 日月光財報 | 封測稼動率、SiP 新訂單 |
| NVIDIA 財報 | GPU 出貨量、下一代路線圖 |
| Meta/MS/Google 財報 | AI CapEx 金額年增率 |
| 費城半導體指數 (SOX) | 全球半導體景氣溫度計 |

### 重要先行指標：Book-to-Bill Ratio

> [!important] Book-to-Bill (BB ratio) 解讀
> - **> 1.0**：新接訂單 > 出貨量，需求強勁，景氣上行
> - **= 1.0**：供需平衡
> - **< 1.0**：新接訂單 < 出貨量，需求疲弱，景氣下行
>
> 台灣電子業 BB ratio 轉正（>1.0）通常是景氣確認訊號，股價在此前已先行。

## 📋 景氣位置自我評估檢查表

> [!tip] 每季填寫一次，判斷當前景氣位置

**上游指標（勾選符合者）：**
- [ ] 矽晶圓現貨價高於長約價
- [ ] 環球晶長約佔比持續提升
- [ ] 家登 EUV 光罩盒出貨量創高
- [ ] 漢唐廠務工程在手訂單創新高

**中游指標：**
- [ ] 台積電稼動率 > 90%
- [ ] IC設計廠庫存天數 < 70天
- [ ] 台積電法說上修全年展望
- [ ] DRAM 現貨價連續3個月上漲

**下游指標：**
- [ ] 日月光稼動率 > 85%
- [ ] 廣達/緯穎 AI 伺服器月營收年增 > 50%
- [ ] 手機出貨量 YoY > 5%
- [ ] ABF 載板交期超過 12 週

**判讀：**
- 上游指標 ≥ 3 個：景氣底部，可積極布局上游
- 中游指標 ≥ 3 個：景氣復甦，可布局中游
- 下游指標 ≥ 3 個：景氣旺季，持有中注意高點
