# 🔶 中游產業：IC設計 / 晶圓代工 / 記憶體

> [!important] 中游的核心地位
> 中游是供應鏈的**技術核心與價值核心**。台積電一家公司的市值超過整個台灣股市的1/3。  
> 中游的景氣狀態決定整條供應鏈的方向。

---

## 🏭 A. 晶圓代工（Foundry）

### 台積電 (2330) — 全球最先進晶圓代工

| 項目 | 內容 |
|------|------|
| 市場地位 | 全球先進製程市佔率 >90%（3nm以下）|
| 核心產品 | 3nm、5nm、7nm 邏輯製程；CoWoS 先進封裝 |
| 主要客戶 | Apple、NVIDIA、AMD、Qualcomm、Intel |
| 護城河 | 技術領先3-5年、客戶黏著度高、規模優勢 |
| 關鍵指標 | CoWoS 產能、先進製程稼動率、CapEx 指引 |
| 分析詳頁 | [[台積電 2330 完整分析]] |

**台積電製程節點與應用：**

| 製程 | 主要應用 | 代表客戶 |
|------|---------|---------|
| 3nm (N3) | 高端手機AP、PC CPU | Apple A17、M4 |
| 4nm (N4) | AI GPU、伺服器 | NVIDIA H100/H200 |
| 5nm (N5) | 中端手機、PC | Qualcomm 8gen2 |
| 7nm (N7) | 成熟AI、網通 | AMD EPYC（舊） |

### 聯電 (2303) — 成熟製程龍頭

| 項目 | 內容 |
|------|------|
| 製程範圍 | 22nm ～ 0.5μm（成熟製程） |
| 主要客戶 | 聯詠、瑞昱、Marvell、德州儀器 |
| 定位 | 不競爭先進製程，深耕成熟製程利基 |
| 優勢 | 特殊製程（車用、射頻）差異化 |
| 風險 | 中國成熟製程產能過剩造成價格壓力 |

### 世界先進 (5347)

| 項目 | 內容 |
|------|------|
| 製程 | 0.11μm ～ 0.5μm 特殊製程 |
| 主力產品 | PMIC、顯示驅動IC代工 |
| 特色 | 聯電轉投資，股東結構穩定 |
| 客戶 | 聯詠（面板驅動IC）、瑞鼎（AMOLED）|

> [!tip] 晶圓代工選股邏輯
> - **台積電**：跟著 AI GPU 需求走，CoWoS 稼動率是核心觀察指標
> - **聯電**：觀察消費性電子庫存去化，成熟製程利用率
> - **世界先進**：跟 AMOLED 手機滲透率連動（瑞鼎、聯詠是其客戶）

## 💡 B. IC 設計（Fabless）

### 聯發科 (2454) — 台灣最大 IC 設計公司

| 項目 | 內容 |
|------|------|
| 核心產品 | 手機AP（天璣系列）、WiFi/BT晶片、車用 |
| 主要客戶 | 小米、OPPO、vivo、三星（中低端）|
| 市場地位 | 全球手機AP市佔第2（僅次高通）|
| 成長動能 | WiFi 7、衛星通訊、車用 ADAS |
| 風險 | 中國手機市場復甦不穩定、高通競爭 |
| 觀察重點 | 天璣旗艦滲透率、中國手機出貨量 |

### 瑞昱 (2379) — 網路晶片霸主

| 項目 | 內容 |
|------|------|
| 核心產品 | 網路晶片（乙太網路）、音訊晶片 |
| 市場地位 | PC音訊晶片市佔 >80%；網路晶片前3大 |
| 成長動能 | AI PC 帶動高速網路需求、新型乙太網路交換機 |
| 景氣敏感 | 中，需求相對穩定（PC + 網路基建）|

### 聯詠 (3034) — 面板驅動IC龍頭

| 項目 | 內容 |
|------|------|
| 核心產品 | LCD 面板驅動IC（DDIC）、TDDI |
| 主要客戶 | 京東方、華星光電（中國面板廠）|
| 市場地位 | 全球 LCD DDIC 市佔前2 |
| 風險 | LCD 滲透率見頂、中國競爭者崛起（奕斯偉）|
| 觀察重點 | AMOLED 驅動IC是否成功轉型 |

### 瑞鼎 (3592) — AMOLED 驅動IC 精品

| 項目 | 內容 |
|------|------|
| 核心產品 | AMOLED 小尺寸驅動IC（SMDDI）、PMIC |
| 主要客戶 | 三星 Display、天馬、維信諾 |
| 市場地位 | 全球 AMOLED 驅動IC 市佔前2 |
| 成長動能 | AMOLED 滲透率提升、摺疊螢幕、車用OLED |
| 分析詳頁 | [[瑞鼎 3592 完整分析]] |

### 矽力杰 (6415) — 電源管理IC 高毛利王

| 項目 | 內容 |
|------|------|
| 核心產品 | 電源管理IC（PMIC）、DC-DC轉換器 |
| 主要客戶 | 華為、小米（中國手機）|
| 特色 | 毛利率 50%+，IC設計中的高毛利品種 |
| 風險 | 中國客戶佔比高，地緣政治風險大 |

## 💾 C. 記憶體

### 南亞科技 (2408) — 台灣 DRAM 唯一標的

| 項目 | 內容 |
|------|------|
| 產品 | DDR4、DDR5 DRAM |
| 股東結構 | 台塑集團（長期持股）|
| 市場地位 | 全球 DRAM 前5（第4-5名）|
| 景氣敏感度 | **極高**，DRAM 現貨價每週波動 |
| 觀察指標 | DXI（DRAM現貨價指數）、Micron/三星庫存 |
| 特性 | 景氣循環最明顯的標的，急漲急跌 |

### 華邦電 (2344) — NOR Flash 龍頭

| 項目 | 內容 |
|------|------|
| 產品 | NOR Flash（BIOS晶片、IoT、車用）|
| 市場地位 | 全球 NOR Flash 第2大 |
| 特色 | NOR Flash 應用穩定（每台電腦都有）|
| 成長動能 | 車用電子需求成長（每台車需多片NOR Flash）|
| 風險 | 旺宏（同業）競爭、定價能力弱 |

## ⚡ 中游選股切入時機

> [!tip] 晶圓代工（台積電）
> - AI GPU 訂單能見度強化（NVIDIA 下一代路線圖確認）
> - CoWoS 產能利用率 > 90%
> - 台積電法說上修全年展望

> [!tip] IC 設計
> - 下游客戶庫存天數降至正常水位（<70天）
> - 手機出貨量年增率轉正
> - AI PC 出貨比重突破 20%

> [!warning] 記憶體（南亞科）
> 最適合在 DRAM 現貨價觸底、季報虧損最嚴重時買入。
> 通常比費城半導體指數提前 1-2 季落底。
