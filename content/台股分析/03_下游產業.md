# 🔽 下游產業：封裝測試 / ABF載板 / 系統組裝

> [!important] 下游的特性
> 下游是供應鏈的**落後指標**。景氣確認時，下游系統廠的出貨量才開始明顯成長。  
> 優點：確定性高；缺點：股價通常已提前反映。
> **封測與 ABF 載板**因為 AI 需求，已從落後指標轉為同步指標。

---

## 📦 A. 封裝測試（OSAT）

### 日月光投控 (3711) — 全球第1大封測廠

| 項目 | 內容 |
|------|------|
| 市場地位 | 全球封測市佔率 **第1**（約30%）|
| 核心服務 | 晶片封裝（SiP、FC-BGA）、測試、系統整合 |
| 主要客戶 | 高通、聯發科、Apple、NVIDIA |
| 成長動能 | SiP 模組（Apple AirPods/Watch）、AI 晶片封測 |
| 護城河 | 規模最大、客戶黏著度高、SiP 技術領先 |
| 風險 | 資本密集、CoWoS 部分搶走台積電封裝市佔 |
| 毛利率 | 15-22% |

**日月光服務範圍：**

```
裸晶（Die）
    ↓ [打線/覆晶/SiP 封裝]
封裝完成品
    ↓ [功能測試/老化測試]
合格IC → 交給客戶（組裝廠/模組廠）
```

### 京元電子 (2449) — 獨立測試廠龍頭

| 項目 | 內容 |
|------|------|
| 核心業務 | **純測試**（不做封裝），IC功能驗證 |
| 主要客戶 | 聯發科、聯詠、矽力杰（台灣IC設計大廠）|
| 特色 | 台灣最大獨立 IC 測試廠 |
| 優勢 | 不與客戶競爭（不做設計），深受 IC 設計廠信任 |
| 成長動能 | 高端晶片測試時間增長（AI/先進製程測試更複雜）|
| 毛利率 | 25-35% |

> [!tip] 封測選股邏輯
> 封測廠稼動率（Utilization Rate）是景氣晴雨表：
> - 稼動率 >85%：景氣旺季，考慮買進
> - 稼動率 <75%：景氣修正期，等待底部

## 🖥️ B. ABF 載板（AI 伺服器關鍵零件）

> [!important] ABF 載板的重要性
> ABF（味之素堆積薄膜）載板是 CPU / GPU 晶片和主機板之間的關鍵橋接。  
> **沒有 ABF 載板，NVIDIA GPU 就無法運作。**  
> AI 爆發使 ABF 載板需求從 2022 年起大幅成長。

### 南電（南亞電路板）(8046) — ABF 載板龍頭

| 項目 | 內容 |
|------|------|
| 核心產品 | ABF 基板（Substrate）|
| 主要客戶 | Intel、NVIDIA、AMD |
| 市場地位 | 全球 ABF 載板前3大 |
| 成長動能 | AI GPU（H100/H200/B200）用量爆增 |
| 護城河 | ABF 載板技術門檻高、擴產週期長（2-3年）|
| 毛利率 | 20-35%（AI旺季可達35%+）|
| 風險 | 資本支出龐大、良率提升是關鍵 |

**ABF 載板在 GPU 中的位置：**

```
NVIDIA GPU 晶片（台積電 CoWoS 封裝）
         ↓
    ABF 載板（南電/欣興供應）
         ↓
    PCB 主板（伺服器主機板）
         ↓
    系統整合（廣達/緯穎組裝）
```

## ⚡ 下游選股切入時機

> [!tip] ABF 載板（南電/欣興）
> AI 伺服器新訂單確認、NVIDIA 下一代 GPU 路線圖宣布時買入
> 是**最具AI Beta**的下游標的，波動大但彈性也大

> [!tip] AI 伺服器組裝（廣達/緯穎）
> 超大規模雲端客戶資本支出指引上修（AWS/Azure/GCP/Meta財報）
> NVIDIA 出貨量數據確認時跟進

> [!warning] 傳統系統組裝（鴻海/仁寶）
> 這類公司跟隨消費電子景氣，AI Beta 較低
> 適合景氣復甦期買入，旺季（Q3-Q4）前出場
