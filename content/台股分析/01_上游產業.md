# 🔺 上游產業：原材料 / 設備 / 製程耗材

> [!important] 上游的戰略意義
> 上游是供應鏈的**先行指標**。當晶圓廠開始擴產，最先受惠的是設備商和材料商。  
> 台積電每一個新廠投資都會帶動數百億的上游採購。

---

## 🔮 上游產業結構

```
台積電/聯電（晶圓廠）的建廠與量產需求
        │
        ├── 矽晶圓（每片晶圓的原料）
        ├── EUV / ArF 光罩盒（光刻製程保護）
        ├── 特殊製程氣體（蝕刻/沉積）
        ├── 廠務工程（超純水/廢氣處理/機電）
        └── 半導體設備（蝕刻機/沉積機/量測機）
              ↑ 多為外商（ASML/Applied/Lam）
              台灣廠主要做：設備零件、維護服務
```

## 🏭 矽晶圓

### 環球晶圓 (6488)

| 項目 | 內容 |
|------|------|
| 全球排名 | 第3大矽晶圓供應商 |
| 主要客戶 | 台積電、英飛凌、意法半導體 |
| 產品 | 8吋、12吋矽晶圓 |
| 關鍵優勢 | 長約佔比高（≥80%），價格波動小 |
| 風險 | 晶圓廠擴產放緩時，新增需求消失 |
| 景氣位置 | 先行指標，台積電擴廠計畫直接影響訂單 |

### 台勝科 (3532)

| 項目 | 內容 |
|------|------|
| 定位 | 小型矽晶圓廠，主供台積電和聯電 |
| 特色 | 台積電持股，具備供應穩定性優勢 |
| 風險 | 客戶集中度高，高度依賴台積電 |

> [!tip] 矽晶圓選股邏輯
> 當台積電宣布新廠投資（如日本、美國、歐洲廠），矽晶圓廠在**2-4季後**開始受惠於新增長約需求。
> 環球晶的長約結構使其比現貨市場更穩定，但也意味漲價彈性受限。

## 🔬 EUV 製程關鍵耗材

### 家登精密 (3680)

| 項目 | 內容 |
|------|------|
| 核心產品 | EUV 光罩盒（SMIF Pod）、光罩傳送盒 |
| 市場地位 | 全球唯一可量產 EUV 光罩盒的台灣廠商 |
| 主要客戶 | 台積電（高度依賴，約佔營收 60%+）|
| 競爭者 | 信越（日）、Entegris（美）|
| 關鍵邏輯 | 台積電每條 EUV 產線都必須用家登光罩盒 |
| 風險 | 客戶集中、台積電議價能力強、技術門檻雖高但競爭者在追趕 |

> [!warning] 高度依賴單一客戶
> 家登約 60% 以上營收來自台積電。台積電資本支出縮減時，家登首當其衝。
> 但台積電先進製程持續推進（2nm、1.4nm），EUV光罩盒需求長期向好。

## 🏗️ 半導體廠務工程

### 漢唐集成 (2404)

| 項目 | 內容 |
|------|------|
| 核心業務 | 半導體廠務統包工程（機電、超純水、廢氣）|
| 主要客戶 | 台積電（南科/竹科/中科擴廠）|
| 商業模式 | 接台積電新廠工程→完工後移轉→持續維保 |
| 景氣敏感 | 高，直接受台積電資本支出影響 |
| 競爭優勢 | 深度綁定台積電，是台積電信任廠商 |

> [!tip] 台積電資本支出追蹤
> 台積電每年法說公布年度資本支出（CapEx），漢唐的未來12個月營收能見度極高。
> 台積電 CapEx 400-500億美元時，漢唐訂單滿載。

## ⚡ 上游選股切入時機

> [!tip] 最佳進場時機
> **景氣底部確認後、台積電宣布擴廠計畫後的 1-2 季內**
>
> 訊號：
> - 台積電 CapEx 指引上修
> - 台積電法說提到 CoWoS / 先進製程滿載
> - 環球晶長約洽談增加
> - 家登接到新 EUV 光罩盒訂單

> [!warning] 避開時機
> 台積電下修資本支出指引 → 上游公司股價通常提前修正 1 季
