# 🤖 AI 伺服器供應鏈（獨立章節）

> [!important] 為什麼單獨建立 AI 供應鏈章節
> AI 伺服器供應鏈不同於傳統科技供應鏈：
> - 需求能見度更長（雲端廠商多年資本支出計畫）
> - 零組件技術門檻更高（CoWoS、液冷、ABF）
> - 景氣循環性較低（AI算力需求持續增長）
> - 台灣廠商在此供應鏈的參與程度極高

## 🔷 台積電 (2330) — CoWoS 先進封裝核心

> [!important] CoWoS 是 AI 時代台積電的第二成長引擎

| 項目 | 內容 |
|------|------|
| CoWoS 全名 | Chip on Wafer on Substrate |
| 技術意義 | 將 GPU + HBM 記憶體整合在同一基板上，大幅縮短傳輸距離 |
| 台積電角色 | **全球唯一**能做 NVIDIA H100/H200/B200 CoWoS 的廠商 |
| 產能現況 | 2024年 CoWoS 月產能約 30,000 片，2025年目標翻倍 |
| 毛利率 | CoWoS 封裝毛利率高於傳統邏輯代工 |
| 瓶頸 | CoWoS 產能是 AI 晶片出貨最大瓶頸 |

**CoWoS 產能擴張時間線：**

| 時間 | 月產能目標 |
|------|---------|
| 2023年底 | ~12,000 片 |
| 2024年底 | ~30,000 片 |
| 2025年目標 | ~60,000 片 |
| 長期目標 | 持續擴產（受限於 SoIC 廠房建設）|

> [!warning] CoWoS 供需缺口
> NVIDIA 的 AI GPU 需求遠超台積電 CoWoS 產能。
> 每次台積電宣布 CoWoS 擴產計畫，都是 AI 供應鏈股票的催化劑。

## 🔴 奇鋐科技 (3017) — 液冷散熱龍頭

| 項目 | 內容 |
|------|------|
| 核心產品 | 液冷散熱模組、冷板（Cold Plate）|
| 主要客戶 | 廣達、緯穎、超微（AMD）|
| 市場地位 | 台灣液冷散熱最具規模廠商 |
| 技術 | Direct Liquid Cooling（DLC）、浸沒式冷卻 |
| 成長動能 | AI 伺服器功耗暴增（H100: 700W → B200: 1200W）|
| 毛利率 | 35-50% |

**AI 晶片功耗進化（液冷必要性）：**

```
A100 GPU: ~400W → 風冷可行
H100 GPU: ~700W → 開始需要液冷輔助
B200 GPU: ~1200W → 液冷幾乎是必須
未來架構: ~2000W+ → 浸沒式冷卻可能必要
```

## 🎯 AI 供應鏈選股追蹤指標

> [!tip] 關鍵追蹤數據（按月追蹤）
> 1. **NVIDIA 出貨量**：每季財報，GPU 出貨量年增率
> 2. **超大規模雲端 CapEx**：Meta / Microsoft / Google / Amazon 季報資本支出
> 3. **台積電 CoWoS 稼動率**：台積電法說中提及的先進封裝滿載情況
> 4. **廣達/緯穎月營收**：每月10日公布，AI 伺服器佔比趨勢
> 5. **南電 ABF 載板報價**：通常每季議價一次

> [!warning] AI 供應鏈主要風險
> 1. NVIDIA 新品延遲（B200 → B300 交接期）
> 2. 超大規模雲端 CapEx 超支導致縮減
> 3. AI 需求不及預期（ROI 遭質疑）
> 4. 中國國產 GPU 突破（替代 NVIDIA）
> 5. 台積電 CoWoS 良率問題
